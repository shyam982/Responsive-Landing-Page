# Responsive Landing Page

A simple responsive landing page built with HTML, CSS, and JavaScript.

## Features
- Responsive layout
- Clean styling
- Simple interactive behavior

## Files
- index.html - Main structure of the page
- style.css - Styling and responsive design
- script.js - JavaScript interactions

## How to Run
1. Open index.html in your browser, or
2. Use a local development server to preview the page

You can customize the content, colors, and sections to match your project.
